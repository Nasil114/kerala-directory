#ഓൺലൈൻകേരളം

പ്രത്യേക വിഭാഗങ്ങൾക്കുള്ള പദ്ധതികൾ

വയോജനങ്ങൾ, ഭിന്നശേഷിക്കാർ, സാമ്പത്തിക സഹായ പദ്ധതികൾ.

Elderly Differently Abled Financial Aid

പ്രത്യേക വിഭാഗങ്ങൾക്കുള്ള പദ്ധതികൾ

A. വയോജനങ്ങൾക്കായി (For Elderly)

1. വയോമിത്രം (Vayomithram): സൗജന്യ മരുന്നും ആരോഗ്യ പരിശോധനയും വീട്ടിലെത്തിക്കുന്ന പദ്ധതി.
2. സായംപ്രഭ (Sayamprabha): വയോജനങ്ങൾക്കായി ഡേ-കെയർ സെന്ററുകൾ ഒരുക്കുന്ന പദ്ധതി.
3. വാതിൽപടി സേവനം (Doorstep Service): മരുന്നുകളും പെൻഷനും വീട്ടിലെത്തിക്കുന്ന സംവിധാനം.
4. സക്കീർ (SACRED): വയോജനക്കാർക്ക് തൊഴിൽ കണ്ടെത്താനുള്ള പോർട്ടൽ.
5. പ്രശാന്തി (Prashanthi): മുതിർന്ന പൗരർക്കുള്ള പോലീസ് ഹെൽപ്പ് ലൈൻ സംവിധാനം.

B. ഭിന്നശേഷിക്കാർക്കായി (For Differently Abled)

1. അനുയാത്ര (Anuyathra): ഭിന്നശേഷിക്കാർക്കായുള്ള സമഗ്ര പരിരക്ഷാ പദ്ധതി.
2. ശ്രുതിതരംഗം (Sruthitharangam): കേൾവിയില്ലാത്ത കുട്ടികൾക്കുള്ള കോക്ലിയർ ഇംപ്ലാന്റേഷൻ പദ്ധതി.
3. ശുഭയാത്ര (Shubhayathra): സഹായ ഉപകരണങ്ങൾ (മുച്ചക്ര വാഹനം, വീൽചെയർ) നൽകുന്ന പദ്ധതി.
4. സ്പെക്ട്രം (Spectrum): ഓട്ടിസം ബാധിതർക്കുള്ള പ്രത്യേക തെറാപ്പി സെന്ററുകൾ.
5. ഹസ്തദാനം (Hasthadanam): ഭിന്നശേഷിയുള്ള കുട്ടികൾക്ക് ഒറ്റത്തവണയായി നൽകുന്ന ധനസഹായം.

C. മറ്റ് സാമ്പത്തിക സഹായ പദ്ധതികൾ (Financial Assistance)

1. ആശ്വാസകിരണം (Aswasakiranam): കിടപ്പിലായ രോഗികളെ പരിചരിക്കുന്നവർക്കുള്ള പ്രതിമാസ ധനസഹായം.
2. സമാശ്വാസം (Samaswasam): വൃക്കരോഗികൾക്കും ഹീമോഫീലിയ ബാധിതർക്കും നൽകുന്ന പെൻഷൻ.
3. സ്നേഹപൂർവ്വം (Snehapoorvam): അഗതികളായ കുട്ടികളുടെ പഠനത്തിനായുള്ള സാമ്പത്തിക സഹായം.
4. വിജയാമൃതം (Vijayamritham): മികച്ച വിജയം നേടുന്ന ഭിന്നശേഷി വിദ്യാർത്ഥികൾക്കുള്ള പുരസ്കാരം.
5. കാരുണ്യ ബെനവലന്റ് ഫണ്ട് (KBF): മാരക രോഗങ്ങളുടെ ചികിത്സയ്ക്കായി നൽകുന്ന ധനസഹായം.
